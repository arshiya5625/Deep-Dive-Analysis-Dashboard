# Deep-Dive Analysis & Interactive Dashboarding

## Project Overview

This project focuses on performing advanced data analysis on a customer sales dataset and building an interactive dashboard for better business understanding.

The objective was to define key performance indicators (KPIs), perform a deep-dive analysis using customer segmentation, and present insights through a Power BI dashboard.

---

## Project Objectives

- Define core business KPIs  
- Perform deep-dive analysis on customer behavior  
- Segment customers based on spending patterns  
- Build an interactive dashboard  
- Present insights for decision-making  

---

## Dataset Description

The dataset includes customer demographic and transaction data:

- Customer_ID  
- Customer_Age  
- Gender  
- City  
- Purchase_Date  
- Purchase_Amount  

This dataset was cleaned and prepared in previous tasks.

---

## Core KPIs

The following key metrics were defined:

- Total Revenue – Total income generated from all purchases  
- Total Customers – Number of customers in the dataset  
- Average Purchase Value – Average amount spent per customer  
- Revenue by City – Revenue contribution by location  
- Monthly Revenue Trend – Revenue variation over time  

---

## Deep-Dive Analysis (Customer Segmentation)

Customers were segmented based on purchase amount into:

- Low Spending Customers  
- Medium Spending Customers  
- High Spending Customers  

### Key Insights

- Medium spending customers are highest in number  
- High-value customers contribute a major portion of revenue  
- Revenue distribution varies across cities  
- Customer behavior differs across segments  

---

## Interactive Dashboard

An interactive dashboard was created using Power BI.

### Dashboard Features

- KPI cards for quick insights  
- Revenue by City (bar chart)  
- Monthly Revenue Trend (line chart)  
- Gender Distribution (pie chart)  
- Spending Category Analysis  
- Filters for City, Gender, and Spending Category  

The dashboard allows users to interact with the data and explore insights dynamically.
